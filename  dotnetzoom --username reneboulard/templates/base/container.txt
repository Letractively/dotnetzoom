1. Login to the Portal as the Administrator
2. Select Admin / File Manager
3. Select Upload New File(s)
4. Browse to the container.zip file
5. Select Add
6. Make sure the Decompress ZIP files option is checked
7. Select Upload
8. Got to the Home tab
9. select the Edit Module icon ( to the left of the module title for any module )
10. Scroll down to Module Container and select Edit Container
